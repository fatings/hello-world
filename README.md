# hello-world

## Estrutura

```text
.
├── index.html
├── script.js
├── styles.css
└── games/
    └── sudoku.html
```

## Ficheiros

- `index.html` — página principal da landing page; contém a estrutura HTML, navegação, secções de conteúdo e referência aos ficheiros CSS e JavaScript.
- `styles.css` — ficheiro de estilos da página principal; define layout, cores, tipografia, responsividade e componentes visuais.
- `script.js` — ficheiro JavaScript da página principal; atualiza o ano no rodapé e roda mensagens de texto na zona principal.
- `games/sudoku.html` — página independente com um jogo de Sudoku, incluída na pasta `games`.
